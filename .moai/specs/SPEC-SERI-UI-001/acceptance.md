# SPEC-SERI-UI-001 Acceptance Criteria

<!-- TAG: SPEC-SERI-UI-001 -->

## Criteria

1. [ ] 1400px 이상 뷰포트에서 "매출분석" 헤더 텍스트가 한 줄로 표시된다
2. [ ] 매출 분석 페이지 하단에 불필요한 빈 공간(100px 초과)이 없다
3. [ ] 320px ~ 2560px 전 범위에서 레이아웃이 정상 렌더링된다
4. [ ] 기존 다른 페이지의 스타일에 영향 없음

## Test Cases

### TC-01: 헤더 텍스트 정상 표시

```
Given: 매출 분석 페이지에 접근한다
When:  브라우저 뷰포트를 1400px, 1920px, 2560px로 각각 설정한다
Then:  "매출분석" 헤더 텍스트가 한 줄로 표시된다
And:   텍스트가 세로로 쌓이지 않는다
```

### TC-02: 하단 빈 공간 제거

```
Given: 매출 분석 페이지에 접근한다
When:  페이지를 최하단까지 스크롤한다
Then:  마지막 콘텐츠 요소와 페이지 하단 사이에 100px 초과의 빈 공간이 없다
```

### TC-03: 모바일 레이아웃 리그레션

```
Given: 매출 분석 페이지에 접근한다
When:  뷰포트를 320px, 768px로 설정한다
Then:  모든 콘텐츠가 정상 렌더링되고 오버플로우가 발생하지 않는다
```
