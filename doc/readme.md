# sajang.ai

> 사장님의 생존을 분석하는 AI 경영 비서

---

## 1. 프로젝트 개요

sajang.ai는  
엑셀/구글시트 기반 가게부 사용자를 위한  
자동연동 + AI 경영 분석 SaaS이다.

기존 소상공인 관리 서비스가  
"데이터 집계" 중심이라면,  

sajang.ai는  
"의사결정과 생존 확률 분석"에 초점을 둔다.

---

## 2. 문제 정의 (Problem)

현재 소상공인 회계 관리 방식:

- 엑셀 수기 입력
- 유료 템플릿 사용
- 카드매출 위주 SaaS 사용

문제점:

- 수동 입력 피로도
- 미래 예측 없음
- 현금흐름 관리 부재
- 생존 확률 판단 불가
- 의사결정 시뮬레이션 부재

---

## 3. 핵심 가치 제안 (Value Proposition)

### 기존 서비스
- 매출 정리
- 세무 대응
- 정부지원금 안내

### sajang.ai
- 생존 확률 점수
- 현금 고갈 예측
- 인건비 위험 감지
- 업종 평균 대비 분석
- 의사결정 시뮬레이션

---

## 4. MVP 범위

### 4.1 기본 기능

- 매출 입력
- 매입 입력
- 고정비 입력
- 월별 손익 자동 계산
- 기본 대시보드

### 4.2 자동화 (1차)

- CSV 업로드 자동 분류
- 카드/배달 매출 자동 정규화
- 반복 지출 자동 카테고리화

### 4.3 AI 분석 (1차)

- 손익 위험 점수
- 고정비 비율 경고
- 인건비 과다 감지
- 3개월 현금 고갈 예측

## 6. 사용자 경험 전략 (UX)

원칙:

- 엑셀 느낌 유지
- 입력 최소화
- 첫 화면 3개 숫자만 입력

초기 입력:

- 월 매출
- 임대료
- 인건비

→ 즉시 리스크 점수 출력
