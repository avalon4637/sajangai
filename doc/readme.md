# sajang.ai

> 하루 330원, 점장 한 명 — 점장이, 답장이, 세리, 바이럴이 매장 운영을 알아서 챙겨드립니다

---

## 1. 서비스 개요

소상공인이 실제 점장을 고용한 것처럼 가게 운영을 맡기는 **개인화 AI Agent 서비스**.
사장님이 카카오톡/앱으로 대화하면 AI 점장이 매출/리뷰/마케팅/재무를 함께 관리한다.

### 핵심 가치 제안

> "하루 330원, 점장 한 명"
> 커피 한 잔 값으로 AI 점장을 고용한다는 감각

### 차별화 포인트

- **캐시노트** → "오늘 매출 312,000원입니다" (정보 제공)
- **세일즈랩** → "리뷰 답글을 작성했습니다" (단일 작업)
- **AI 점장** → "매출이 12% 하락 중입니다. 원인은 배달 지연 리뷰 급증이고, 오늘 답글 달고 단골 3명에게 할인 쿠폰 문자 보낼까요?" (종합 판단 + 행동)

## 2. 에이전트 구성

```
+-------------------------------------+
|          점장 (Supervisor)           |
|  통합 오케스트레이션 / 리포트 발송    |
|  에이전트 답변 정리 / 이상 알림      |
+----------+----------+---------------+
| 답장이   | 세리     | 바이럴        |
| (리뷰)   | (재무)   | (마케팅)      |
|          |          |               |
| 리뷰수집 | 매출분석 | 문자발송      |
| 감성분석 | 정산관리 | SNS제안       |
| 자동답글 | 이상감지 | 재방문유도    |
+----------+----------+---------------+
```

### 에이전트별 역할

| 에이전트 | 역할 | 주요 기능 | 데이터 소스 |
|---------|------|---------|-----------|
| 점장 | Supervisor | 전체 조율/에이전트 답변 정리/일간 리포트 발송/이상 알림 | 전체 통합 |
| 답장이 | 리뷰 매니저 | 리뷰 수집/분석/자동 답글/감성 분석 | 배민/쿠팡이츠/요기요/네이버 |
| 세리 | 매출 분석가 | 매출 집계/정산 분석/이상 감지 | 카드매출/배달앱 매출/홈택스 |
| 바이럴 | 마케터 | 재방문 유도/문자 발송/SNS 제안 | 고객 데이터/매출 패턴 |

## 4. 데이터 연동 - 하이픈(Hyphen) 단일화

### 왜 하이픈인가

직접 스크래핑 엔진을 만들면 플랫폼 UI 변경 시마다 장애가 발생한다.
하이픈은 배달앱 리뷰/매출부터 카드매출까지 이미 상품화된 검증된 서비스다.

### 하이픈 연동 상품 목록

| 상품 | 제공 데이터 | 방식 |
|------|-----------|------|
| 배달의민족 | 매출/정산/리뷰/주문내역/통계 | 스크래핑 API |
| 쿠팡이츠 | 매출/정산/리뷰/주문내역/통계 | 스크래핑 API |
| 요기요 | 매출/정산/리뷰/주문내역 | 스크래핑 API |
| 배달앱 통합 (주문+리뷰) | 배민+쿠팡+요기요+네이버 리뷰 통합 | 스크래핑 API |
| 카드매출 (여신금융협회) | 승인/매입/입금 내역 | 스크래핑 API |
| 홈택스 | 매입매출/세금계산서/현금영수증 | 스크래핑 API |

> 코드에프(카드매출) 별도 계약 불필요 - 하이픈 하나로 통일

### 연동 아키텍처

```
사장님 온보딩 (1회)
    |
배민/쿠팡/요기요/카드 ID/PW 입력
    |
하이픈 Connected ID 발급 (원본 PW 암호화 저장)
    |
+-------------------------------+
|         하이픈 API             |
|  배달앱 리뷰     OK            |
|  배달앱 매출     OK            |
|  카드 매출       OK            |
|  홈택스 매출     OK            |
+-------------------------------+
    |
AI 점장 컨텍스트 주입
    |
사장님에게 통합 리포트
```

### 하이픈 API 호출 설계

**수집 주기 (유료 사용자 기준)**

```
오전 6시 자동 수집:
  배민 리뷰 신규분       1회
  쿠팡이츠 리뷰 신규분   1회
  요기요 리뷰 신규분     1회
  배민 어제 매출         1회
  카드 어제 매출         1회
                         ----
일 5회 x 30일 = 150회

월 1회 수집:
  월 정산 (배민/쿠팡/요기요/카드)  4회
                                   ----
월 총 호출 = 약 154회/사장님
```

**무료 체험 사용자 - 수집 제한**

```
무료 체험: 주 1회 수집 -> 월 8회 -> 하이픈 비용 약 240원
유료 구독: 일 1회 수집 -> 월 154회 -> 하이픈 비용 약 2,300~4,600원
```

## 5. 사업자 인증 설계

### 인증 흐름

```
[체험 시작]
배민/요기요/카드 계정 연결
    |
하이픈에서 사업자번호 자동 추출
    |
국세청 사업자 진위확인 API (무료)
    |
DB에 사업자번호 + 연결일 저장
    |
체험 시작

[재가입 시도 차단]
동일 사업자번호 감지
    |
"이미 체험이 완료된 사업장입니다. 구독으로 계속 이용하세요."

[유료 전환]
결제 시점에 핸드폰 인증 추가
(카드 결제 정보가 본인 확인 역할)
```

### 남용 리스크 평가

- 사업자번호는 변경 불가
- 배민/카드 계정 자체가 실명 연동
- 새 사업자 개설 비용 > 체험 가치
- **결론: 사업자번호 1회 제한으로 충분**

### 국세청 진위확인 API

```
POST https://api.odcloud.kr/api/nts-businessman/v1/status
Request: { "b_no": ["123-45-67890"] }
Response: { "data": [{ "b_no": "1234567890", "b_stt": "계속사업자", "tax_type": "부가가치세 일반과세자" }] }
```

비용: 무료 (공공 데이터 포털 API 키 발급)

## 7. 7일 무료 체험 설계

### 날짜별 경험 설계 (핵심)

체험 7일을 그냥 주는 게 아니라 날마다 "없으면 안 되겠다"는 순간을 설계한다.

| 일차 | 경험 | 임팩트 |
|------|------|--------|
| D+1 | 연결 완료 + 어제 매출/미답글 리뷰 수 첫 리포트 | "오 이게 되네" |
| D+2 | 점장 첫 인사이트 - "배달 느림 키워드 2건, 답글 달아드릴까요?" | AI가 알아서 분석 |
| D+3 | 재무 에이전트 - "이번 주 배민 매출 지난주 대비 12% 하락" | 몰랐던 사실 발견 |
| D+5 | 마케팅 에이전트 - "단골 3명이 2주째 안 오셨어요. 재방문 문자 보낼까요?" | 내가 못 챙긴 걸 챙겨줌 |
| D+6 | 1주일 운영 리포트 자동 생성 + 구독 전환 CTA | 전환 의지 자극 |
| D+7 | "오늘 자정에 점장과의 계약이 종료됩니다" | 긴급성 -> 전환율 피크 |

### 전환 CTA 문구

```
X "프리미엄으로 업그레이드"
X "구독을 시작하세요"

O "점장 고용 계약 연장하기"
O "월 9,900원으로 점장 계속 고용하기"
O "하루 330원, 점장 한 명"
```

## 8. 수익성 시뮬레이션

### 체험 전환 수익성

```
체험 1명당 원가:
  하이픈 API (7일 x 5회 x 30원)  = 약 1,050원
  Claude API (7일 x 약 1~2회)    = 약 2,000원
  -------------------------------------------
  합계                            = 약 3,050원

전환율 31% = 손익분기점
전환율 40% = 흑자 전환
```

### 구독자 규모별 월 수익

| 구독자 | 하이픈 단가 | 하이픈 월 비용 | Claude 비용 | 서버 | 월 순이익 | 마진율 |
|--------|-----------|-------------|-----------|------|---------|-------|
| 100명 | 30원/건 | 46만원 | 20만원 | 10만원 | 약 23만원 | 23% |
| 1,000명 | 15원/건 | 230만원 | 200만원 | 30만원 | 약 530만원 | 53% |
| 5,000명 | 8원/건 | 616만원 | 750만원 | 100만원 | 약 3,484만원 | 70% |

> 1,000명부터 흑자 구조 안정화, 5,000명부터 대량 계약으로 마진율 급등

## 10. 토큰 비용 최적화 (캐싱 전략)

### 메모리 계층화

```
Hot Memory  (Redis)      오늘 대화, 현재 세션
Warm Memory (DB)         최근 30일 요약본
Cold Memory (Vector DB)  전체 히스토리 (검색 시에만)
```

### 3가지 핵심 기법

1. **Anthropic Prompt Caching** - 가게 프로필(시스템 프롬프트) 캐싱 -> 토큰 비용 90% 절감
2. **대화 요약 압축** - 10턴 대화(~3,000 토큰) -> 요약본(~150 토큰), Haiku로 저렴하게 압축
3. **Redis 반복 캐싱** - "오늘 매출 어때?" 같은 반복 질문 -> LLM 호출 없이 캐시 반환

### 절감 효과 비교

| 방식 | 토큰/대화 | 월 비용 (1천명/10회/일) |
|------|---------|----------------------|
| 풀 컨텍스트 무제한 | ~15,000 | ~450만원 |
| 선택적 로드 | ~4,000 | ~120만원 |
| 요약 + 캐싱 전체 적용 | ~2,000 | ~60만원 |

## 11. MVP 개발 로드맵

### 1단계 - 정보 연결 + 대시보드 (4주)

- 하이픈 API 연동 (배민/카드/요기요)
- 국세청 사업자 인증 연동
- 기본 대시보드 (매출/리뷰 현황)
- 7일 체험 플로우 구현

### 2단계 - AI 에이전트 활성화 (4주)

- 리뷰 에이전트: 자동 수집/감성 분석/답글 생성
- 재무 에이전트: 일별 매출 리포트/이상 감지
- 카카오 알림톡 연동

### 3단계 - 마케팅 에이전트 + 과금 (3주)

- 마케팅 에이전트: 재방문 유도/문자 발송
- 결제 시스템 (카카오페이/토스)
- 구독 전환 자동화 플로우

### 4단계 - 통합 점장 고도화 (지속)

- 통합 점장 Supervisor 개선
- 업종별 특화 프롬프트 팩
- 네이버 플레이스 리뷰 연동 (파트너십)
