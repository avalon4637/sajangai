# Roadmap

sajang.ai 개발 로드맵

---

## Phase 1: MVP Foundation (4주)

### Sprint 1 — 기본 인프라 + 데이터 입력 (Week 1-2)

- [ ] Supabase 프로젝트 생성 및 DB 스키마 마이그레이션
- [ ] 인증 시스템 (이메일 로그인/회원가입)
- [ ] 사업장 등록/관리
- [ ] 매출 입력 UI (엑셀 느낌 테이블)
- [ ] 매입/지출 입력 UI
- [ ] 고정비 등록 UI

### Sprint 2 — KPI + 대시보드 (Week 3-4)

- [ ] KPI 계산 엔진 연동
  - 매출총이익률 (gross_margin)
  - 인건비 비율 (labor_ratio)
  - 고정비 비율 (fixed_cost_ratio)
  - 생존 점수 (survival_score)
- [ ] 월별 손익 자동 계산 + 캐싱
- [ ] 메인 대시보드 UI
  - 핵심 숫자 3개 카드 (매출, 순이익, 생존점수)
  - 월별 추이 라인 차트
  - 비용 구조 파이 차트

**Phase 1 완료 목표**: 수동 입력으로 기본 손익 분석 가능한 MVP

---

## Phase 2: 자동화 + AI (4주)

### Sprint 3 — CSV 자동화 (Week 5-6)

- [ ] CSV 업로드 UI (드래그 앤 드롭)
- [ ] CSV 파싱 및 자동 분류 엔진 연동
- [ ] 카드/배달 매출 자동 정규화
- [ ] 반복 지출 자동 카테고리화
- [ ] 업로드 미리보기 + 수정 UI
- [ ] 업로드 이력 관리

### Sprint 4 — AI 분석 (Week 7-8)

- [ ] Claude API 연동 (Vercel AI SDK)
- [ ] AI 분석 리포트 생성
  - 손익 위험 점수 해석
  - 고정비 비율 경고
  - 인건비 과다 감지
  - 3개월 현금 고갈 예측
- [ ] AI 인사이트 대시보드 위젯
- [ ] 대화형 AI 상담 (채팅 UI)

**Phase 2 완료 목표**: CSV 업로드로 자동 입력 + AI 경영 분석

---

## Phase 3: 시뮬레이션 + 수익화 (4주)

### Sprint 5 — 의사결정 시뮬레이션 (Week 9-10)

- [ ] 시뮬레이션 엔진 연동
  - 직원 증감 시뮬레이션
  - 매출 변동 시뮬레이션
  - 임대료 변동 시뮬레이션
- [ ] 시뮬레이션 UI (슬라이더/버튼 기반)
- [ ] Before/After 비교 시각화
- [ ] 시뮬레이션 저장 기능

### Sprint 6 — 수익화 + 폴리싱 (Week 11-12)

- [ ] 구독 플랜 설정 (Free / Pro / Premium)
- [ ] 결제 연동 (Stripe 또는 토스페이먼츠)
- [ ] Pro 기능 게이팅
- [ ] 온보딩 플로우 (3개 숫자 입력 → 즉시 결과)
- [ ] 랜딩 페이지
- [ ] SEO 최적화

**Phase 3 완료 목표**: 유료 전환 가능한 완성된 서비스

## Phase 4: 확장 (이후)

- [ ] 업종 평균 비교 데이터 축적 및 비교 기능
- [ ] Monte Carlo 기반 고급 생존 확률 모델
- [ ] PDF 경영 리포트 생성 및 다운로드
- [ ] 카드사/배달앱 API 직접 연동
- [ ] 모바일 최적화 (PWA)
- [ ] 금융 확장 (대출 평가 연계)
- [ ] 권리금닷컴 연동 (거래 데이터 ↔ 운영 데이터)

## 수익 모델

| 플랜 | 가격 | 기능 |
|------|------|------|
| **Free** | 무료 | 수동 입력, 기본 손익표, 생존 점수 |
| **Pro** | 월 19,000~39,000원 | CSV 자동 연동, AI 분석 리포트, 시뮬레이션 |
| **Premium** | 별도 협의 | 업종 평균 비교, 고급 생존 모델, PDF 리포트 |
