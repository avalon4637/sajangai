## 카드 표시 위치

모든 카드는 **점장 홈 모바일 피드**에 시간순으로 표시:
```
점장 홈 (모바일)
├─ KPI 스트립 (고정)
├─ BriefingRichCard (매일)
├─ InsightCard × N (수시)
├─ ReviewAlertCard (수시)
├─ CashflowWarningCard (조건부)
├─ PeriodReportCard (주간/월간)
├─ SubscriptionAlertCard (이벤트)
├─ 추천 질문
└─ 채팅 입력 (고정)
```
