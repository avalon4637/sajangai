Real user feedback collected 2026-04-06:

Many small business owners operate 2-3 businesses simultaneously. Current system assumes 1 user = 1 business.

**Why:** This is a common pattern in Korean F&B industry. Single owner, multiple stores (different brands or locations).
**How to apply:** Need SPEC for multi-business support — business switcher UI, per-business data isolation, subscription model consideration (per-business or per-account pricing).
