type: feedback

바이럴 에이전트의 원래 컨셉은 SNS 콘텐츠 생성이었으나, 현재 디자인은 이탈 방지/재방문 마케팅으로 설계됨.

**Why:** 사용자가 바이럴의 핵심은 SNS 컨텐츠 생성이라고 명시. 현재 디자인(이탈 고객 + 카카오 메시지)은 다른 방향.

**How to apply:** 바이럴 페이지는 추후 별도 SPEC으로 SNS 콘텐츠 생성 컨셉으로 재설계. 현재 3페이지(점장/세리/답장이)를 우선 구현.
